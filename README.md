# Hello-Claude-SpringBoot

A minimal "Hello World" Spring Boot application created to explore development workflows with Claude.

## Overview
This project serves as a sandbox for testing and learning. Its primary purpose is to provide a clean, functional starting point to experiment with how AI assistants like Claude can help in writing, testing, and maintaining Spring Boot applications.

## Features
- **Spring Boot**: Minimal setup with the `spring-boot-starter-web` dependency.
- **REST API**: Includes a simple endpoint to verify the application is running.
- **Developer Sandbox**: Designed to be modified, broken, and refactored while interacting with Claude.

## Prerequisites
- **Java**: JDK 21 or higher
- **Build Tool**: Maven 
- **IDE**: IntelliJ IDEA
